# 🎯 NEXUS 2026 — Event Landing Page

> A premium, fully responsive conference landing page built with 
> vanilla HTML, CSS & Bootstrap 5. Zero frameworks, zero dependencies.

## 🌐 Live Demo
🔗 [View Live →](https://nexus-2026.vercel.app)

---

## ✨ Features

| Section | Description |
|---|---|
| 🦸 Hero | Full-screen with animated live countdown timer |
| 📊 Stats Band | Animated counters — speakers, attendees, sessions |
| 🎤 Speakers | 8 speaker cards with hover bio + live search filter |
| 📅 Schedule | 3-day tabbed agenda with keynote, panel, workshop tags |
| 🎟️ Tickets | 3-tier pricing cards (Explorer / Innovator / Visionary) |
| 📍 Venue | Location section with image and travel info |
| 📝 Register | Full form with validation + success state |
| 📧 Footer | Newsletter signup + social links |

## 🛠️ Tech Stack

- **HTML5** — Semantic, accessible markup
- **CSS3** — Custom properties, animations, glassmorphism
- **Bootstrap 5.3** — Grid, tabs, modals, form validation
- **Bootstrap Icons** — Icon system throughout
- **Vanilla JavaScript** — Countdown timer, live search, form handling
- **Google Fonts** — Bebas Neue · Cormorant Garamond · DM Sans
